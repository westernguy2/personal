Graph API and Trip-finder (Java)

Created a graph package using Java that allowed users to create a graph data structure for their own use.

Included in the API was a general traversal algorithm as well as breadth-first and depth-first traversal algorithms.

Also included in the API is an algorithm that determines the shortest path between two nodes in the graph. The algorithm uses A* and Dijkstra's to find the shortest path.

In addition to the graph API, I created a trip-finder that uses the shortest path feature of the API to find the best set of directions between two locations.

The example above shows the shortest path from Berkeley to Santa Cruz, with a waypoint in San Francisco.
